Bootstrap Responsive Grid

Farklı ekran boyutlarında kolonların nasıl değiştiğini öğreniyoruz

Bootstrap Breakpoint Mantığı

Bootstrap grid sistemi farklı ekran boyutlarına göre kolonları değiştirebilir. Buna Responsive Grid denir.

  • col- → tüm ekranlar
  • col-sm- → küçük ekranlar (≥576px)
  • col-md- → tablet (≥768px)
  • col-lg- → laptop (≥992px)
  • col-xl- → büyük ekran (≥1200px)
