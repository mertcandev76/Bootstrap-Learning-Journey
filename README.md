# Bootstrap-Learning-Journey

Bu depo, Bootstrap framework’ünü öğrenirken yaptığım 
pratik çalışmalarımı içermektedir.

Bu projenin amacı, Bootstrap bileşenlerini, yerleşim sistemlerini (layout) ve yardımcı sınıflarını adım adım uygulayarak frontend geliştirme becerilerimi geliştirmektir. Her klasör belirli bir ders konusunu temsil eder ve ilgili Bootstrap kavramlarını gösteren basit HTML örnekleri içerir.

Bu öğrenme süreci boyunca aşağıdaki önemli Bootstrap konularına odaklanılmaktadır:

- Container ve sayfa yerleşim yapısı
- Grid sistemi
- Flexbox yardımcı sınıfları
- Tipografi ve metin yardımcı sınıfları
- Renkler, kenarlıklar ve boşluklar (margin & padding)
- Position kullanımı
- Bootstrap bileşenleri (cards, alerts, buttons, navbar vb.)
- Etkileşimli bileşenler (modal, dropdown, tooltip, carousel)
- Form elemanları ve doğrulama (validation)

Tüm örnekler, Bootstrap sınıflarının gerçek projelerde nasıl çalıştığını daha iyi anlayabilmek için basit HTML dosyaları kullanılarak oluşturulmuştur.

Bu depo aynı zamanda Bootstrap öğrenme sürecimdeki günlük ilerlememi takip ettiğim bir öğrenme günlüğü niteliği de taşımaktadır.
